# Ashfall PBR textures

Source: [Poly Haven](https://polyhaven.com), license **CC0** (public domain,
no attribution required — credited here anyway). 1K JPG maps (diffuse,
normal GL, roughness) fetched via `api.polyhaven.com`:

- `burned_ground_01` — the village floor
- `damaged_plaster` — house walls
- `broken_brick_wall` — brick stumps, chimneys, the well
- `black_painted_planks` — charred beams, the cart

Higher resolutions exist at the source if a hero shot ever needs them;
swap the `_1k` suffix in AshfallCinematic.tsx.
